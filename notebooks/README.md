# Trading Analysis Notebooks

This directory contains Jupyter notebooks for:
- Data visualization and exploration
- Technical indicator development
- Strategy backtesting
- Performance analysis

## Setup

1. Install dependencies:
```bash
uv sync --group dev
```

2. Start Jupyter:
```bash
jupyter notebook
# or
jupyter lab
```

## Notebooks

- `basic_visualization.ipynb` - Basic charting with indicators
- `indicator_development.ipynb` - Developing and testing custom indicators
- `backtesting.ipynb` - Strategy backtesting examples
